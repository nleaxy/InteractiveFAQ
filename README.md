# Конструктор интерактивных FAQ с поиском по синонимам и голосовым вводом

Платформа для создания баз знаний и разделов вопросов-ответов с интеллектуальным поиском. Позволяет находить информацию даже при неточных формулировках запросов благодаря обработке синонимов и нечеткому поиску.

---

## 🛠 Технологический стек

### Бэкенд (Backend)
- **Язык:** Python 3.10+
- **Фреймворк:** FastAPI (асинхронный веб-фреймворк)
- **Валидация данных:** Pydantic v2
- **База данных:** SQLite / PostgreSQL (с SQLAlchemy в качестве ORM)

### Фронтенд (Frontend)
- **Сборщик и среда:** Vite + React (TypeScript)
- **Стилизация:** Tailwind CSS + shadcn/ui
- **Работа с сервером:** Axios + TanStack Query (React Query)
- **Голосовой ввод:** Web Speech API (распознавание на стороне клиента)
